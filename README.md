# 欢迎使用 OpenResty Lua 代码补全插件

## 功能 (Features)

* 支持 require 文件跳转
* 支持 _load 文件跳转
* 支持变量定义跳转
* 支持全局库（api）代码补全
* 本地变量（upvalue）代码补全
* 支持有限制的 api 声明类型推导
* 支持 dao 模块类型推导
* 支持引用模块类型推导
* 支持本地变量类型推导
* 支持文档大纲

## 依赖 (Dependences)

* [luaparse](https://github.com/fstirlitz/luaparse)
* [luacheck](https://github.com/mpeterv/luacheck)
